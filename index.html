MEET APP

An app that will allow users to search for a city and get a list of events hosted in that city. This is a serverless, PWA application built with React. It uses the OAuth2 protocol for authentication and AWS Lambda as its authorization server. Also, this app makes use of a service worker to allow the app to display previously stored data when the device is offline.
